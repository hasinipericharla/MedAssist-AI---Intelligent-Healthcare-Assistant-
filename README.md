# 🩺 MedAssist AI — Intelligent Healthcare Assistant

An AI-powered healthcare platform that predicts probable diseases from 
user-reported symptoms and explains *why* it made that prediction — 
going beyond a typical "input symptoms, get disease" ML project.

⚠️ **Disclaimer:** This project is built for educational purposes. 
Predictions are generated by a machine learning model and are 
**not a substitute for professional medical diagnosis or advice.**

---

## 🎯 What This Project Does

Most student ML projects stop at "predict X from Y." MedAssist AI goes 
further by combining prediction with **explainability, personalization, 
and a full product experience**:

- Predicts the most likely disease from a list of symptoms
- Shows the **top 3 possible diseases** with confidence scores
- Explains **which symptoms contributed how much** to each prediction
- Gives **personalized health recommendations** based on the user's 
  age, BMI, and medical history — not generic advice
- Generates a **downloadable PDF medical report**
- Tracks prediction history and shows analytics over time
- Lets users **chat** to ask follow-up questions about their prediction
- Finds **nearby hospitals and clinics** based on the user's location
- Supports **voice input** for symptoms via speech-to-text
- Sends **medicine reminders** at scheduled times

## ✨ Features

| Feature | Description |
|---|---|
| Disease Prediction | Decision Tree & Random Forest models compared |
| Explainable AI | Per-symptom contribution breakdown for each prediction |
| Personalized Recommendations | Based on user's health profile (BMI, age, allergies, history) |
| PDF Medical Report | Auto-generated report with symptoms, prediction, and suggestions |
| Health Chat Assistant | Ask follow-up questions like "why did I get Flu?" |
| Analytics Dashboard | Prediction trends, disease distribution, model accuracy |
| Authentication | Secure login/signup with JWT |
| Prediction History | Searchable, filterable past predictions |
| Nearby Hospital Finder | Suggests nearby hospitals/clinics after prediction |
| Voice Assistant | Speak symptoms instead of typing, via speech-to-text |
| Medicine Reminder | Set reminders for prescribed medicine times |
